‚úÖ With Source Maps (sourcemap: true)
When an error occurs at AboutView-Dzqdte5y.js:1:137:

1. Browser sees //# sourceMappingURL=AboutView-Dzqdte5y.js.map
2. Fetches the .map file (only when DevTools open)
3. Looks up column 137 in the mappings
4. Finds: "This came from AboutView.vue line 2, column 2 (_hoisted_1)"
5. Shows you: AboutView.vue:2:2 (_hoisted_1)
6. Displays original source code when you click!
